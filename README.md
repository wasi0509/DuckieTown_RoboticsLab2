# RIS Lab II – DuckieTown Project

## Overview
This repository contains the work developed for **Robotics and Intelligent Systems (RIS) Lab II**, focusing on applying robotics concepts to a real robotic platform using DuckieTown.
---

## Objectives

- Apply **ROS (Robot Operating System)** in a real-world robotics environment  
- Develop and integrate **robotic software modules**  
- Handle **real-time system constraints**  
- Practice **technical and scientific documentation**

---

1. Robotics software architectures  
2. Use existing ROS libraries and packages  
3. Develop functionalities in robotic simulation environments  
4. Implement solutions on real robotic systems (Duckiebot)  
5. Collaborate effectively in a team-based robotics project  

---

## DuckieTown Platform
DuckieTown (Duckietown) is a robotics platform for autonomous driving education and research. It includes:

- A miniature smart city environment  
- Autonomous robots (Duckiebots)  
- Real-world tasks such as:
  - Lane following  
  - Traffic navigation  
  - Intersection handling  
  - Obstacle avoidance  

---

## Technologies Used
- ROS (Robot Operating System)  
- Python / C++  
- Duckietown framework
